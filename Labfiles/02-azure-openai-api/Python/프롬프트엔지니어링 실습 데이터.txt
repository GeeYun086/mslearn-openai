아래 내용을 번역해 주세요.
A quantum computer is a computer that exploits quantum mechanical phenomena. On small scales, physical matter exhibits properties of both particles and waves, and quantum computing takes advantage of this behavior using specialized hardware. Classical physics cannot explain the operation of these quantum devices, and a scalable quantum computer could perform some calculations exponentially faster.

 PG - Case Breakdown 예시
-----------------------

< 기본 프롬프트 >

윤규는 30초 만에 짚라인을 타고 내려갈 수 있으며, 정상으로 다시 올라가는 데 5분이 걸린다. 윤규가 17분 동안 몇 번이나 짚라인을 탈 수 있을까?

< Case Breakdown & CoT >

- 윤규는 30초 만에 짚라인을 타고 내려갈 수 있으며, 정상으로 다시 올라가는 데 5분이 걸린다. 윤규가 짚라인을 탄 시간은?
- 윤규가 짚라인을 세 번 더 타는데 걸리는 시간과 소요 시간은?
- 지금까지 윤규가 짚라인을 탄 횟수와 총 소요 시간은?


-----------------------
 SDK - 퓨샷 예시 1
-----------------------

# 퓨샷
입력 : "배송이 너무 늦어서 화가 납니다. 당장 환불을 원합니다."
출력 : "배송지연 - 고객님의 배송 지연 문제로 환불 절차를 시작해야 합니다."

입력 : "주문한 상품과 다른 제품이 배송되었습니다."
출력 : "오배송 - 주문 상품과 다른 제품이 배송되었습니다. 교환 절차를 시작하세요."

---

(실습용 프롬프트는 자유롭게 입력하세요)
상품이 파손된 상태로 도착했습니다. 빠른 교환 부탁드립니다.

 SDK - 퓨샷 예시 2

# 퓨샷
입력: "회의에서 A안과 B안을 논의했으며, 최종 A안 채택입니다. 추가적으로 일정 조율 필요."
출력: "결정사항: A안 채택, 일정 조율 필요."

입력: "프로젝트 진행 중 인력 부족과 예산 초과가 지적되었고, 해결책으로 신규 인력 채용과 예산 재조정이 필요합니다."
출력: "결정사항: 신규 인력 채용, 예산 재조정."

---

(실습용 프롬프트는 자유롭게 입력하세요)
회의에서 고객 요구사항 반영과 일정 단축이 주요 안건이었습니다. 이를 위해 전략 수정 TF를 만들기로 했습니다.

-----------------------
 SDK - CoT 예시
-----------------------

# 퓨샷

입력: "배송 지연으로 불만입니다. 주문한 상품이 아직 도착하지 않았어요."
CoT:
 1. 고객이 배송 지연에 대해 불만을 표시함.
 2. 주문 내역과 배송 이력이 확인되어야 함.
 3. 지연 원인은 물류 문제 또는 재고 부족일 가능성이 있음.
최종 출력: "즉시 주문 내역과 배송 상태를 확인하고, 고객에게 예상 배송일과 함께 보상 조치를 안내하세요."

입력: "상품이 오염된 채로 왔어요. 고객으로서 매우 실망했습니다."
CoT:
 1. 상품 상태에 대한 불만이 핵심.
 2. 오염 원인은 포장 부실 또는 생산 결함일 가능성이 있음.
 3. 교환 또는 환불이 시급함.
최종 출력: "상품의 오염 상태를 확인 후, 교환 또는 환불 절차를 즉시 진행하고 고객에게 사과 메시지를 발송하세요."

---

(실습용 프롬프트는 자유롭게 입력하세요)
주문한 제품이 파손되어 도착했습니다. 환불이나 교환 모두 가능한가요?
